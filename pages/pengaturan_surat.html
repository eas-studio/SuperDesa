Pengaturan Surat

Atur format penomoran, jenis huruf, dan tampilan footer untuk semua surat.

Format Penomoran Surat

Gunakan kode berikut:
[nomor] untuk nomor urut surat.
[kode_surat] untuk kode unik setiap jenis surat.
[bulan_romawi] untuk bulan dalam angka Romawi (I, II, ...).
[tahun] untuk tahun saat ini (contoh: 2025).

Tampilan Surat

Kop surat akan selalu menggunakan font Arial.
